Multi-Sport Performance Analytics Dashboard

An interactive data analytics dashboard that analyzes performance trends across Football, Formula 1, and MotoGP using real-world datasets. This project focuses on extracting actionable insights through advanced metrics and visualizations.

📊 Overview

This platform enables users to explore historical sports data, evaluate team and player performance, and uncover trends using a unified analytics interface.

Unlike prediction-based systems, this project emphasizes data analysis, visualization, and business-style insights.

🚀 Features
📈 Performance Trends
Win rates, goals scored, lap times, standings progression
Season-wise and time-series analysis
🧠 Advanced Analytics
Consistency score (variance-based performance)
Rolling averages (last N matches/races)
Win rate & podium frequency
⚔️ Comparative Analysis
Head-to-head comparison of teams/drivers
Cross-season performance evaluation
🌍 Multi-Sport Support
⚽ Football (club-level match data)
🏎️ Formula 1 (driver & constructor stats)
🏍️ MotoGP (rider performance trends)
🎛️ Interactive Dashboard
Dynamic filters (team, driver, season, sport)
